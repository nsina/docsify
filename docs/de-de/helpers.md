# Dokumentationshelfer

docsify erweitert die Markdownsyntax, um deine Dokumente besser lesbar zu machen.

## Wichtiger Inhalt

Wichtiger Inhalt wie:

```markdown
!> **Zeit** ist Geld, mein Freund!
```

wird wie folgt gerendert:

!> **Zeit** ist Geld, mein Freund!

## Generelle Tipps

Generelle Tipps wie:

```markdown
?> *TODO* unit test
```

wird wie folgt gerendert:

?> *TODO* unit test
